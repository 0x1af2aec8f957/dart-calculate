Dart Calculate

This project enable to make computations on decimal numbers without loosing precision like double operations.
For instance :

import 'package:calculate';

print(compute('0.1 0.1 *'));
如果你不习惯逆波兰表达式，推荐使用dart-decimal处理精度问题。
